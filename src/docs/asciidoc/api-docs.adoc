= 개요

== 지원하는 Method(Verb)

|===
| 행위 | 사용법

| `GET`
| 특정 리소스를 조회합니다.

| `POST`
| 새로운 리소스를 생성합니다.

| `PUT`
| 특정 리소스의 모든 정보를 수정합니다.

| `PATCH`
| 특정 리소스의 일부 정보를 수정합니다.

| `DELETE`
| 특정 리소스를 삭제합니다.

| `OPTIONS`
| 현제 url가 지원하는 모든 method를 응답합니다.

| `Head`
| 요청에 대한 Header 정보만 응답합니다.
|===
